## What Happens Next?

1. Click **Create a Bug Ticket**.
2. Complete the bug report form.
3. Submit the ticket.
4. You will receive a ticket ID for tracking.

---

## Before You Submit

Please have the following information ready:

- Viewline version
- Operating System
- Steps to reproduce
- Expected result
- Actual result
- Screenshots or screen recording
- Log files (if available)
